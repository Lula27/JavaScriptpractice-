Global Objects: Date Object (And Time)

To use dates, create an instance of the Date object.
Specify the time and date you want to represent.
In JavaScript, dates are stored as numbers: number of milliseconds since mindnight on January 1, 1970.
Date & time determined by computer clock. Dependent on your internal clock being correct and where (what time zone) a user is in!

METHOD DESCRIPTION EXAMPLE
getDate() / setDate() Returns/sets day of the month (1-31) 1
toDateString() Returns "date" as a human-readable string 2
toString() Returns string representing the specified date 3
getTimezoneOffset() Returns time zone offset in mins for locale 4
getTime() / setTime() Returns number of millisecs. since 1/1/1970, 00:00:00 UTC (Coordinated Universal Time) and a negative number for any time before. 5
toTimeString() Returns "time" as a human-readable string 6
getFullYear() / setFullYear() Returns / sets the year (4 digits) 7
getMonths() / setMonths() Returns / sets the months(0-11) 8
getDay() Returns the day of the week (0-6) 9
getHours() / setHours() Returns / sets the hour (0-23) 10
getMinutes() / setMinutes() Returns / sets the minutes (0-59) 11
getSeconds() / setSeconds() Returns / sets the seconds(0-59) 12
getMilliseconds() / setMilliseconds() Returns / sets the milliseconds (0-999) 13
